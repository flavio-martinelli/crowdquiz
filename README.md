# CrowdQuiz

Real-time quiz and poll system for groups. The host runs it on a laptop, everyone else joins from their phone by scanning a QR code.

## Setup

Python 3.11+ required.

```bash
git clone git@github.com:flavio-martinelli/crowdquiz.git
cd crowdquiz
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn server:asgi_app --host 0.0.0.0 --port 8000
```

Open `http://localhost:8000/host` on the host machine.

## Making it accessible to others

The host runs the server locally. To give players a public URL to join from anywhere, use Cloudflare Tunnel.

Install it:
```bash
brew install cloudflared
```

Then open a second terminal and run:
```bash
cloudflared tunnel --url http://localhost:8000
```

It will print something like `https://random-words.trycloudflare.com`. This URL is randomly generated by [Cloudflare Tunnel](https://developers.cloudflare.com/cloudflare-one/connections/connect-networks/do-more-with-tunnels/trycloudflare/) and disappears as soon as you stop the command (Ctrl+C), so keep that terminal open for the whole game.

Paste the URL into the field at the top of the host lobby and press Set. The QR code will update and players can scan it to join.

## Adding a quiz

Create a `.json` file in the `quizzes/` folder. It will appear automatically in the host lobby.

```json
{
  "title": "My Quiz",
  "description": "Optional description",
  "questions": [
    {
      "type": "quiz",
      "text": "What is the capital of France?",
      "options": ["Berlin", "Madrid", "Paris", "Rome"],
      "correct": 2,
      "time_limit": 20
    },
    {
      "type": "poll",
      "text": "What's your favourite season?",
      "options": ["Spring", "Summer", "Autumn", "Winter"],
      "time_limit": 30
    }
  ]
}
```

`type: quiz` shows a correct answer and gives points. `type: poll` just shows the vote distribution, no scoring. `correct` is the index of the right answer (0-based). `time_limit` is in seconds.
